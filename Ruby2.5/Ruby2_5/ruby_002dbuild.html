2.1.2.1 ruby-build

ruby-build

rbenv

ruby-build is a plugin for rbenv (see section rbenv that allows you to compile and install different versions of Ruby into arbitrary directories. ruby-build can also be used as a standalone program without rbenv. It is available for OS X, Linux, and other UNIX-like operating systems.
