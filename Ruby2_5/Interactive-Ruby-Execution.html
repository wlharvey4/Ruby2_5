2.4.9.1 Interactive Ruby Execution

Interactive Ruby code is entered using the irb tool.

Once irb is installed, launch it as follows: 

$ irb
irb(main):001:0>

Now, we can begin to execute Ruby code: 

irb(main):001:0> puts 'Hello Ruby'
Hello Ruby
=> nil
irb(main):002:0>
