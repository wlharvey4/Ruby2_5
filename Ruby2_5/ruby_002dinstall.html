2.1.2.2 ruby-install

ruby-install version manager chruby version switcher

ruby-install

chruby

ruby-install allows you to compile and install different versions of Ruby into arbitrary directories. There is also a sibling, chruby (see section chruby), which handles switching between Ruby versions. It is available for OS X, Linux, and other UNIX-like operating systems.
