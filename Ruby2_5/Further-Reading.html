2.7 Further Reading

Programming Ruby

by Marek Hulan, et al., contains succinct advanced information about Ruby. See section Programming Ruby by Hulan. (See item ProgrammingRuby in Bibliography.)

Pragmatic Programmer’s Guide — The Pickaxe (aka Programming Ruby)

by Dave Thomas and Andy Hunt. The place where just about every Ruby beginner starts. See The Pickaxe aka Programming Ruby - The Pragmatic Programmer’s Guide.

The Ruby Way

by Hal Fulton. When you’re reasonably familiar with Ruby, but want to learn more and see how to accomplish all sorts of tasks with it, then http://therubyway.io, is the logical next step.
