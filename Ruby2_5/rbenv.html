2.1.3.2 rbenv

rbenv

ruby-build

rbenv allows you to manage multiple installations of Ruby. It does not support installing Ruby, but there is a popular plugin named ruby-build (see section ruby-build) to install Ruby. Both tools are available for OS X, Linux, or other UNIX-like operating systems.
