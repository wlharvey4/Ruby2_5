2.2 Ruby Help Tools

The RDoc system includes the command-line tools rdoc and ri for generating and displaying online documentation.

ri (Ruby Index) and RDoc (Ruby Documentation)1 are a closely related pair of tools for providing documentation about Ruby programs. ri and rdoc are standalone programs; you run them from the command line. The Ruby ri tool is used to view the Ruby documentation off-line. RDoc produces HTML and online documentation for Ruby projects.
