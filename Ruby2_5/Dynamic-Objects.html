4.7.2 Dynamic Objects

Sending A Message To An Object

Every time you invoke a method, you are sending a message to an object. Most of the time, these messages are hard-coded as in a static language, but they need not always be. We can write code that determines at runtime which method to call. The send() method will allow us to us a Symbol to represent a method name. __SEND__ is an alias. public_send() is used to send a message to only public methods.
