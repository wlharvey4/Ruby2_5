A.2.8 apiutil Makefile Target

Add a target for running apiutil.awk in the Makefile. Note that the directory ./bin is created in the @initial_setup environment (see section Initial Setup) and is made executable by a @post_create command (see section Post Create), both of which were added by the TexiwebJR program.

<Makefile—Utility Targets> ≡ 

# apiutil.awk
#############
.PHONY : apiutil
apiutil : 
    bin/apiutil.awk Ruby2_5.twjr

This chunk is also defined in Makefile---Utility-Targets-2.

This chunk is called by {Makefile}; see its first definition at Makefile.
